What is SVG?

  • SVG stands for Scalable Vector Graphics
  • SVG is used to define graphics for the web
  • SVG is a W3C Recommendation

The HRML svg tag/element is a container for SVG grphaics. SVG has serveral methods for drawing paths, boxes, circles, text and graphic images.
